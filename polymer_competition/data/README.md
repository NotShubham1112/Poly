# Data Directory

Immutable input data files. **Never modify files in this directory.**

## Files

| File | Description |
|---|---|
| `train.csv` | Training set with columns: `id`, `SMILES`, `Tg` (or target property) |
| `test.csv` | Test set with columns: `id`, `SMILES` |
| `splits.pkl` | Cross-validation indices (generated by `features/build_features.py`) |

## Expected Format

### train.csv

```
id,SMILES,Tg
1,*CCO*,193.15
2,*C(C)C*,253.27
3,*c1ccc(C(=O)O)cc1*,348.50
...
```

### test.csv

```
id,SMILES
1001,*CC(C)C*
1002,*CO*
...
```

## Notes

- The `*` character in SMILES denotes the connection point between repeat units.
- For multi-monomer copolymers, separate the monomer SMILES with `.` (e.g., `*CCO*.C(C)*`).
- Use `data/.gitkeep` to preserve the folder structure in version control.
